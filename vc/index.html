    Overview¶

    Auto-Generation

    The content in this page (after Typical Stocks) is auto-generated by the PowerShell module vc which calls Python module mllp (vc_filter).

    The data table is generated by pandas (to_markdown) and rendered by mkdocs-material (data-tables).

    Terminologies

    StockTrend

    The stock trend circle: R1 -> R2 -> R3 -> F1 -> F2 -> F3 -> R1 Other status is sideaway.

    • R1 = "R01" # EMA20 > EMA50 < EMA100 < EMA200
    • R2 = "R02" # EMA20 > EMA50 > EMA100 < EMA200
    • R3 = "R03" # EMA20 > EMA50 > EMA100 > EMA200
    • F1 = "F01" # EMA20 < EMA50 > EMA100 > EMA200
    • F2 = "F02" # EMA20 < EMA50 < EMA100 > EMA200
    • F3 = "F03" # EMA20 < EMA50 < EMA100 < EMA200

    Columns

    Column Description
    0 CGR (Close - Last Open) / (Last Open)
    1 RCR (Close - Open) / (Open)
    2 VPR VR / MP
    3 VDR (VP - VP(-1)) / VP(-1)
    4 VR VDR * (100 - RSI) / max(1, RCR)
    5 MT MACD Type such as GC (Golden Cross), DC (Death Cross)
    6 RT RSI Type such as GC (Golden Cross), DC (Death Cross)
    7 EMAT EMA Type such as EMA10TO50 (EMA10 GC EMA50)
    8 FS "Filter Strategy" which indicates the situation when the Open/Low - Close/High cross over EMA lines. E.g. PC1050 means Open/Low <=EMA10 and Close/High >= EMA50
    9 FSL Stock price range might have multiple FS values such as PC1020, PC1050. FSL is the total number of FS of the price range of the stock today
    10 RATE Calculated with one complex formula by considering several columns: MT, FS, TREND, EMAT, VDR, VPR, FSL
    11 PPO EMA50 - EMA5 / EMA50 * 100
    Analysis Process

    Go through VC filter result records, pickup the candidates and put to the pool. In the pool, only keep those having EMA50 in NIU state.
