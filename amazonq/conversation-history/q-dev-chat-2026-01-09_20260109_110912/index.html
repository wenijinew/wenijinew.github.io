    how to set SPARK_HOME

    To set SPARK_HOME, you have several options depending on your operating system and how you want to configure it:

    Windows¶

    Option 1: Environment Variables (System-wide)¶

    1. Open System Properties → Advanced → Environment Variables
    2. Add new system variable:
    3. Variable name: SPARK_HOME
    4. Variable value: C:\path\to\spark (e.g., C:\spark\spark-3.5.0-bin-hadoop3)

    Option 2: PowerShell (Session-specific)¶

    $env:SPARK_HOME = "C:\path\to\spark"

    Option 3: Command Prompt (Session-specific)¶

    set SPARK_HOME=C:\path\to\spark

    Linux/macOS¶

    Option 1: Shell Profile (Persistent)¶

    Add to ~/.bashrc, ~/.zshrc, or ~/.profile: 

    export SPARK_HOME=/path/to/spark
export PATH=$PATH:$SPARK_HOME/bin

    Option 2: Current Session¶

    export SPARK_HOME=/path/to/spark

    Python Code (Runtime)¶

    import os
os.environ['SPARK_HOME'] = '/path/to/spark'

    Verification¶

    After setting, verify with: - Windows: echo %SPARK_HOME% - Linux/macOS: echo $SPARK_HOME - Python: print(os.environ.get('SPARK_HOME'))

    Make sure to replace /path/to/spark with your actual Spark installation directory.
