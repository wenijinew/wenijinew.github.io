    Domain Management¶

    Go to Google Domain and open the target domain. Then click DNS, in Resource Records, create a new record and fill the below fields:

    • Host name: www
    • Type: CNAME
    • TTL: 3600
    • Data: wenijinew.github.io. if the site is to host on github. ubiquitous-boba-dbb112.netlify.app. if the site is to host on netlify. Note: For netlify, you need to check in the Domain Management of your site. Check Manage domains for your production site for more information.
